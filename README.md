## Setup and Run ES6 code

Install dependency (babel > 7.x)
```
npm i -D @babel/node @babel/core @babel/preset-env
```

add .babelrc
```
{
    "presets": [
        "@babel/env"
    ]
}
```

Run 
```
  npx babel-node array_es6.js
  
  npm run test
```
